Extensiones de Chrome creadas por Davincho Moca.

- Web: https://moca.gt
- Autor: Davincho Moca

## Extensiones

### Div Inspector

Carpeta: `div-inspector/`

Inspector visual para resaltar elementos HTML en una pagina y copiar el elemento seleccionado.

### Div Inspector Modify

Carpeta: `div-modify-inspect/`

Inspector visual editable para seleccionar elementos, moverlos, redimensionarlos y copiar el CSS aplicado junto con informacion del contenedor padre.

## Instalacion local

1. Abre `chrome://extensions`.
2. Activa `Developer mode`.
3. Haz clic en `Load unpacked`.
4. Selecciona la carpeta de la extension que quieres probar:
   - `div-inspector/`
   - `div-modify-inspect/`

## Iconos

Ambas extensiones usan el icono generado en `shared-assets/div-inspector-icon-1024.png`, con tamanos exportados para Chrome en cada carpeta `icons/`.
